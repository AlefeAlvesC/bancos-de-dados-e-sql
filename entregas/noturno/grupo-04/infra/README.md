# Infraestrutura - Grupo 4

## Objetivo

Esta pasta reúne as informações sobre execução, ambiente e publicação da aplicação do Grupo 4.

## Ambiente local utilizado

- Backend: Node.js com Express
- Frontend: React com Vite
- Banco: MySQL

## Execução local

### Backend

```bash
cd entregas/noturno/grupo-04/app/escola-api
npm install
npm start
```

### Frontend

```bash
cd entregas/noturno/grupo-04/app/escola-web
npm install
npm run dev
```

## Configuração do ambiente

Use o arquivo `.env.example` na pasta `entregas/noturno/grupo-04/` como base para criar ou ajustar o arquivo `.env`.

## Observação

Caso o projeto seja publicado em ambiente remoto, esta pasta pode ser atualizada com o endereço final de acesso, print da publicação e instruções de deploy.
